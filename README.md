# [konsol].tr

**[konsol].tr** , herhangi bir konu üzerinde bilgi paylaşımı bulunurken konsol uygulamalarından faydalanan tüm türkçe projeler için ortak bir link paylaşım platformu olarak tasarlanmıştır.
